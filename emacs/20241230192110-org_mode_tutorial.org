* Tables
** Calculations
Note that hitting ~TAB~ on ~|-~ autocompletes the underscores under the header row.

| Calc    | Value |
| (4+4)   |       |
| (+ 4 5) |       |
#+TBLFM: @2$2=(4+4) ;; R2C2
#+TBLFM: @3$2='(+ 4 5) ;; R3C2

You can evaluate these using ~C-c C-c~.
*** Row-column examples
Note that multiple calculations can be separated using ~::~'s.
